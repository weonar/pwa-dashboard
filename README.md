# 📱 PWA Dashboard — Local Sync для твоей сети

Локальный дашборд на Node.js, который работает на телефоне (Termux) и синхронизирует данные со всеми устройствами в локальной сети через WebSocket в реальном времени.

---

## ✨ Что новенького в дизайне

**Современный UI с**:
- 🎨 Красивые градиенты (фиолетовый, розовый, голубой)
- 💫 Плавные анимации и эффекты hover
- 🌙 Тёмная тема с glassmorphism (размытые полупрозрачные фоны)
- ✨ Реалистичные тени и глубина элементов
- 📱 Идеально работает на мобильных и десктопах
- ⚡ Быстрые переходы между экранами (300-400ms)

---

## 🎯 Что это может делать?

### 🌐 **Сайт** — Быстрый доступ
- Встроенный iframe с `https://hearts-together.vercel.app/`
- На весь экран, работает в оффлайне (если загрузить один раз)

### 📋 **Буфер обмена** — Синхронизация текста
- Введи текст на одном устройстве → видишь на всех других
- Синхронизация через WebSocket в реальном времени
- Автосохранение через 2 сек

### 📝 **Заметки** — Два режима
1. **Текст** — простые плоские заметки
2. **Список задач** — с чекбоксами, редактируемые, удаляемые
- Сохраняется в JSON файл
- Синхронизация между устройствами

### 📁 **Файлы** — Управление на HDD
- Загрузка файлов с любого устройства в локальную сеть
- Все файлы сохраняются на USB/HDD, подключенный к телефону
- Скачивание и удаление файлов
- Список с размератором и датой

## 📥 Установка на Termux

### Шаг 1: Обнови пакеты и установи Node.js

```bash
pkg update
pkg install nodejs
```

Проверь версию:
```bash
node --version
npm --version
```

### Шаг 2: Скачай проект и установи зависимости

```bash
# Перейди в нужную папку
cd /sdcard/Download  # или другая папка

# Создай папку проекта
mkdir pwa-dashboard
cd pwa-dashboard

# Если у тебя есть файлы (server.js, package.json и т.д.):
# Скопируй их в эту папку

# Установи зависимости
npm install
```

### Шаг 3: Настрой конфиг (опционально)

Открой файл `server.js` и найди строку в начале:

```javascript
const HDD_PATH = process.env.HDD_PATH || '/sdcard/Download';
```

**Измени путь**, если нужно:
- `/sdcard/Download` — папка Download на телефоне
- `/storage/emulated/0/Download` — альтернативный путь
- Если USB подключен: `/sdcard/UMS/DISK_NAME/` или похожий путь

Чтобы найти правильный путь:
```bash
ls /sdcard/
ls -la /storage/
```

### Шаг 4: Запусти сервер

```bash
npm start
```

Ты должен увидеть:

```
╔════════════════════════════════════════╗
║     🚀 PWA DASHBOARD ЗАПУЩЕН 🚀        ║
╚════════════════════════════════════════╝

📱 Открой в браузере: http://localhost:3000
🌐 Из локальной сети: http://<IP-телефона>:3000

📁 Путь к файлам: /sdcard/Download
💾 Данные сохраняются в: ./data

Нажми Ctrl+C для остановки сервера
```

## 🌐 Узнай IP-адрес телефона

Есть несколько способов:

### Способ 1: В Termux

```bash
ip addr show | grep "inet"
```

Или:

```bash
ifconfig | grep -A1 wlan0
```

Ищи строку типа `inet 192.168.x.x`

### Способ 2: В настройках телефона

- **Android**: Настройки → О телефоне → Статус → IP-адрес
- Или: Настройки → Wi-Fi → Нажми на сеть → Статус

### Способ 3: Из интернета (если роутер доступен)

Зайди на IP роутера (обычно 192.168.1.1 или 192.168.0.1) и посмотри в таблице подключенных устройств.

## 🚀 Подключение с других устройств

Когда сервер запущен на телефоне:

1. **На ПК/ноутбук/планшет** открой браузер
2. Введи адрес: `http://<IP-телефона>:3000`
   - Например: `http://192.168.1.100:3000`
3. Приложение загрузится в браузер

Все устройства будут синхронизированы через WebSocket!

## 📦 PWA — Установка как приложение

Приложение — это полноценная PWA (Progressive Web App).

### На мобильном (Android)

1. Открой в Chrome: `http://<IP-телефона>:3000`
2. Нажми кнопку **меню** (три точки)
3. Выбери **"Установить приложение"** или **"Add to Home screen"**
4. Приложение установится как обычное приложение на рабочий стол

### На ПК (Windows/Mac/Linux)

1. В Chrome (или Edge): `http://<IP-телефона>:3000`
2. В адресной строке справа нажми **значок установки** (пазл или плюс)
3. Нажми **"Установить"**
4. Приложение откроется в отдельном окне без браузера

## 🎨 Дизайн и цветовая схема

### 🎭 Цветовая палитра
| Назначение | Цвета |
|--|--|
| **Градиент 1** (Буфер обмена) | Фиолетовый → Средний фиолетовый |
| **Градиент 2** (Заметки) | Розовый → Красный |
| **Градиент 3** (Файлы) | Голубой → Циан |
| **Фон** | Тёмный синий (0f0f1e → 1a1a2e) |
| **Текст** | Светло-серый |

### 🌟 UI элементы
- **Glassmorphism**: Размытые полупрозрачные фоны (rgba с backdrop-filter)
- **Тени**: Мягкие тени для глубины элементов
- **Скруглённые углы**: Для мягкого, современного вида
- **Hover эффекты**: Поднятие на 3px и изменение цвета
- **Переходы**: Плавные cubic-bezier анимации (300-400ms)

### 📱 Адаптивный дизайн
- **Мобильный** (< 768px): Горизонтальная навигация внизу, свайп между экранами
- **Десктоп** (≥ 768px): Вертикальная навигация слева, клики по кнопкам
- **Скролбар**: Тонкий, прозрачный, стилиизованный

## 📖 Функции интерфейса

### 📱 На мобильном — **Свайп между экранами**
- **Свайп влево** → следующий экран
- **Свайп вправо** → предыдущий экран
- Или кликай по кнопкам на навигации внизу

### 💻 На ПК — **Клик по навигации**
- Сайдбар слева с красивыми кнопками
- Клик по кнопке → плавный переход на экран
- Активная кнопка подсвечивается градиентом

### 🎯 Индикатор подключения
- Индикатор в правом нижнем углу
- 🔴 Красный — отключено
- 🔵 Голубой с пульсацией — подключено к серверу

## 📊 API Endpoints

Если нужно использовать дашборд программно:

### Получить файлы
```
GET /api/files
→ JSON массив файлов с размером и датой
```

### Загрузить файл
```
POST /api/upload
Body: multipart/form-data с файлом
→ { success: true, filename: "..." }
```

### Удалить файл
```
DELETE /api/files/:filename
→ { success: true }
```

### Получить буфер обмена
```
GET /api/clipboard
→ { content: "..." }
```

### Получить заметки
```
GET /api/notes
→ { mode: "text"|"tasks", content: "...", tasks: [...] }
```

### WebSocket сообщения

```javascript
// Отправить текст в буфер обмена
{ type: "clipboard", content: "текст" }

// Отправить заметки
{ type: "notes", data: { mode: "text", content: "...", tasks: [...] } }
```

## 🐛 Решение проблем

### ❌ "npm: command not found"
```bash
pkg install nodejs
```

### ❌ "Адрес уже в использовании"
Сервер уже запущен на порту 3000 или другой процесс его занимает.
```bash
# Заверши сервер нажатием Ctrl+C
```

### ❌ "Не может найти путь к файлам"
Убедись, что пусть в `server.js` правильный:
```bash
ls /sdcard/
ls /storage/emulated/0/
```

### ❌ "Не могу подключиться с другого устройства"
- Проверь, что оба устройства в одной сети Wi-Fi
- Проверь IP адрес телефона
- Убедись, что сервер запущен
- Попробуй отключить брандмауэр на телефоне

### ❌ "Данные синхронизируются с задержкой"
Это нормально — WebSocket работает медленнее всего на мобильных сетях. Попробуй:
- Ближе к роутеру Wi-Fi
- Перезагрузи приложение

## 💡 Советы и трюки

### Замораживание экрана Termux
Termux может отключить приложение, если экран заблокирован. Чтобы этого избежать:

```bash
# Установи WakeLock приложение из Play Store
# Или используй опцию в настройках Termux
```

Или держи экран включённым:

```bash
# Включи режим разработчика на телефоне
# Включи опцию "Оставить включённым"
```

### Автозапуск при перезагрузке
Создай скрипт:

```bash
nano start.sh
```

Добавь:

```bash
#!/bin/bash
cd /sdcard/Download/pwa-dashboard
npm start
```

Сохрани: Ctrl+X → Y → Enter

Запусти:

```bash
chmod +x start.sh
./start.sh
```
